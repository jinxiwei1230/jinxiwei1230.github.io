1.3 数据类型

1.3.1基本数据类型

整数类型

  • byte 8位 1字节
  • short 16位 2字节
  • int 32位 4字节 -long 64位 8字节（定义时在结尾加上L）

浮点数类型

  • float(4字节) （定义时必须在结尾加上F或者f）
  • double(8字节) （定义时可以在结尾加上D或者d,也可以不加）

布尔类型

  • boolean（true，false）

字符类型（2字节）

  • char（\u0000～\uFFFF ）
  1. 一个字符用单引号，一个字符串用双引号
  2. 转义字符
  • 转义字符也是字符，所以将转义字符赋值给其他字符变量时也需要使用单引号

1.3.2引用数据类型

  • 数组
  • 类class
  • 接口interface

1.4变量与常量

  1. 变量
  • 变量声明 [修饰符] 类型 变量 [=表达式]
  1. 常量：只能被赋值一次
  • 常量声明 final 数据类型 常量名称 [=值]（变量名称一般用大写）
  1. 变量的有效范围

  • 成员变量：在类中定义，分为静态变量和实例变量（静态变量能以“类名.静态变量”的方式在其他类中使用）

  • 局部变量：在类的方法中定义的变量，局部变量只在当前代码块中有效

  • 局部变量与成员变量名字可以相同，此时成员变量将被隐藏，想要使用成员变量时需要使用“类名.静态变量”

1.5运算符

  1. 赋值运算符“=”

  2. 算数运算符：+，-，*，/，%(+可用作连接符)

  3. 自增，自减运算符:

  • ++a(–a)表示在使用变量a之前，先使a的值加（减）1
  • a++(a–)表示在使用变量a之后，使a的值加（减）1

  1. 比较运算符：>,<,<=,>=,==,!=

  2. 逻辑运算符：

  • &&(&)逻辑与，左到右（&&短路，&非短路）
  • ||(|)逻辑或，左到右（||短路，|非短路）
  • !逻辑非，右到左
  1. 位运算符：
  • 按位运算：~，&，|，^(按位异或)
  • 移位运算：«,»,»>(无符号右移，0扩展)
  1. 三元运算符
  • num=i>j?100:200
  1. 运算符的优先级

1.6类型转换

1.6.1自动类型转换

  • 表达式自动类型转换：在表达式中，小范围的变量会自动转化为大范围的变量，再参与运算。表达式的最终结果类型由表达式中的最高类型决定。
  • byte,short,char是直接转化为int类型参与运算的

1.6.2强制类型转换

  • (类型名)要转换的值
  • Boolean不能与其他类型数据相互转换

1.7 代码注释

  • 单行注释//
  • 多行注释/* */
  • 文档注释/** */
